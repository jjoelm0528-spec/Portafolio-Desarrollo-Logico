# Portafolio-Desarrollo-Logico
Portafolio de Desarrollo Lógico y Algoritmos

Por: Jonathan Joel Morales Cruz

Este repositorio contiene mi portafolio digital, desarrollado para la asignatura Desarrollo Lógico y Algoritmos de la Universidad Tecnológica de Panamá (UTP).

El objetivo de este portafolio es demostrar la aplicación práctica de los conceptos de lógica de programación, diseño algorítmico y herramientas tecnológicas cubiertos durante el curso.

Acceso Rápido al Portafolio Publicado

El portafolio está desplegado utilizando GitHub Pages para garantizar un acceso rápido y seguro para la evaluación.

Estructura del Portafolio
El proyecto es una aplicación web de una sola página, construida con HTML, CSS (diseño personalizado con inspiración en Pixel Art y Tema Oscuro) y JavaScript para la funcionalidad bilingüe.

El portafolio incluye las siguientes secciones:

Acerca de Mí / Profile: Breve autobiografía, trayectoria académica y lista de aspiraciones en áreas clave de la ingeniería de sistemas (Ciberseguridad, Python, Diseño de Videojuegos, etc.).

Asignaciones del Curso / Course Assignments: Recuento de las 8 entregas y evaluaciones realizadas durante el desarrollo de la materia, incluyendo laboratorios, investigaciones y el proyecto final.
